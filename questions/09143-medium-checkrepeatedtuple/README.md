Implement type ```CheckRepeatedChars<T>``` which will return whether type ```T``` contains duplicated member? 

For example:

```ts
type CheckRepeatedTuple<[1, 2, 3]>   // false
type CheckRepeatedTuple<[1, 2, 1]>   // true
```
