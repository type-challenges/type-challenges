Implement a generic `Last<T>` that returns the last element of an array `T`.

For example:

```ts
type result1 = Last<[1, 2, 3]> // expected to be 3
type result2 = Last<['a', 'b', 'c']> // expected to be 'c'
type result3 = Last<[]> // expected to be never
```
