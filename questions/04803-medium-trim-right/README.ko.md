정확한 문자열 타입이고 끝부분의 공백이 제거된 새 문자열을 반환하는 `Trim<T>`를 구현하십시오.

예시

```ts
type Trimed = TrimRight<'   Hello World    '> // 기대되는 결과는 '   Hello World'입니다.
```
