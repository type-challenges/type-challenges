문자열 인수를 입력받는 String to Union 유형을 구현하세요.
출력은 입력 문자열의 Union type이어야 합니다.

예시:

```ts
type Test = "123"
type Result = StringToUnion<Test> // expected to be "1" | "2" | "3"
```
