> この課題ではTypeScript 4.0が推奨されます

配列 `T` を受け取り、その最後の要素の型を返す汎用的な `Last<T>` を実装してください。

例えば

```ts
type arr1 = ['a', 'b', 'c']
type arr2 = [3, 2, 1]

type tail1 = Last<arr1> // expected to be 'c'
type tail2 = Last<arr2> // expected to be 1
```
